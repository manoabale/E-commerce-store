# E-commerce-store
A simple front‑end e‑commerce demo that displays products, allows users to add items to a cart, and calculates the total price.

🚀 Features

• Product listing
• Add‑to‑cart functionality
• Cart total calculation
• Clean and simple UI


🧩 Tech Stack

• HTML
• CSS
• JavaScript


📂 Project Structure

index.html
style.css
script.js


📸 Preview

Displays a list of products and a cart section with total price.

🧠 How It Works

Products are stored in an array and rendered dynamically. Clicking “Add” pushes the product into the cart array and updates the UI.
